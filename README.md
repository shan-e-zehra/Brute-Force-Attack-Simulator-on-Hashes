# 🔐 Brute Force Attack Simulator on Hashes

A Python-based cybersecurity educational project that simulates brute-force and dictionary attacks on hashed passwords in a safe and ethical environment. The project helps students understand password security, hashing algorithms, salting, and common password-cracking techniques through an interactive graphical user interface.

## 📌 Features

- Generate password hashes using secure hashing algorithms
- Simulate Brute Force attacks
- Simulate Dictionary attacks using custom wordlists
- Password salting support
- Interactive GUI for easy experimentation
- Modular Python architecture for better maintainability
- Designed for cybersecurity education and awareness

## 🛠️ Technologies Used

- Python
- Tkinter (GUI)
- hashlib
- Cryptography Concepts
- Password Hashing
- Brute Force Attack Simulation
- Dictionary Attack Simulation

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/Brute-Force-Attack-Simulator.git
```

2. Navigate to the project folder

```bash
cd Brute-Force-Attack-Simulator
```

3. Install dependencies (if required)

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python gui.py
```

## 📖 How It Works

1. Enter a password or hash.
2. Select a hashing algorithm.
3. Choose either:
   - Brute Force Attack
   - Dictionary Attack
4. Enable or disable password salting.
5. Observe the attack results and compare the effectiveness of different security measures.

## 🎯 Learning Outcomes

- Understanding password hashing
- Importance of password salting
- Difference between brute-force and dictionary attacks
- Secure authentication concepts
- Ethical password-cracking simulations
- Practical Python implementation of cybersecurity concepts
